## 二、验证指标说明

### 1. First-touch 比例

**定义**: 入场后先到 +0.5R 还是先到 -0.5R 的比例

**计算公式**（对于 LONG）:
```
+0.5R = entry_price + 0.5 * (entry_price - stop_loss)
-0.5R = entry_price - 0.5 * (entry_price - stop_loss)
```

**意义**: 衡量信号入场后的初始方向选择

### 2. 可达率分布

**定义**: 24h 内达到 +1R / +2R / +3.5R 的比例

**计算公式**（对于 LONG）:
```
+1R = entry_price + 1.0 * (entry_price - stop_loss)
+2R = entry_price + 2.0 * (entry_price - stop_loss)
+3.5R = entry_price + 3.5 * (entry_price - stop_loss)
```

**意义**: 衡量信号入场后的后续空间

## 五、最终结论

### ❌ 2023 年失效源于后续空间不足

**核心证据**:

1. **更容易先到 -0.5R 而不是 +0.5R**
   - 2023 年先到 -0.5R 比例: 51.8%
   - 2023 年先到 +0.5R 比例: 48.2%
   - 差异: **-3.6%**（负值）

2. **对 +1R / +2R / +3.5R 的可达率显著低于 2024/2025 年**
   - +1R 可达率差异: **+26.9% / +23.0%**
   - +2R 可达率差异: **+24.3% / +21.2%**
   - +3.5R 可达率差异: **+28.2% / +21.1%**

**结论**:
- 2023 年 LONG 信号入场后，更容易先触及 -0.5R（逆行），说明初始方向不利
- 2023 年 LONG 信号入场后，后续空间严重不足，无法达到 +1R / +2R / +3.5R 目标
- **根本原因**: 趋势延续不足，导致后续空间受限

## 八、附录：技术细节

### 验证脚本

**文件**: `scripts/verify_path_level.py`

**核心功能**:
1. 运行 2023/2024/2025 三年回测
2. 获取每笔仓位入场后 24h 的 K 线数据
3. 判断 first-touch（先到 +0.5R 还是 -0.5R）
4. 判断可达率（达到 +1R / +2R / +3.5R 的比例）
5. 按年汇总统计指标
6. 跨年对比分析

### 关键指标计算

**First-touch 判断**:
```python
risk = abs(entry_price - stop_loss)
target_up = entry_price + 0.5 * risk    # +0.5R
target_down = entry_price - 0.5 * risk  # -0.5R

for kline in klines:
    if kline.high >= target_up:
        return "UP_FIRST"
    if kline.low <= target_down:
        return "DOWN_FIRST"
```

**可达率判断**:
```python
risk = abs(entry_price - stop_loss)
target_1R = entry_price + 1.0 * risk
target_2R = entry_price + 2.0 * risk
target_3_5R = entry_price + 3.5 * risk

for kline in klines:
    if kline.high >= target_1R:
        reach_1R = True
    if kline.high >= target_2R:
        reach_2R = True
    if kline.high >= target_3_5R:
        reach_3_5R = True
```
