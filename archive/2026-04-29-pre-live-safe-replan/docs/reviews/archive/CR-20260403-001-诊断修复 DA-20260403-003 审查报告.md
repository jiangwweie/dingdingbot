### ⚠️ 发现的问题

#### 【P1-重要】问题 1: backtest_repository.py 缺少列存在性检查

**位置**: `src/infrastructure/backtest_repository.py` 第 644 行

**问题代码**:
```python
"sharpe_ratio": str(self._str_to_decimal(row["sharpe_ratio"])) if row["sharpe_ratio"] else None,
```

**风险**: 如果数据库未运行迁移脚本，`sharpe_ratio` 列不存在时会抛出 `KeyError`

**修复建议**:
```python
# 方案 1: 使用 getattr 安全访问 (推荐)
"sharpe_ratio": str(self._str_to_decimal(row["sharpe_ratio"])) if row.keys() and row["sharpe_ratio"] else None,

# 方案 2: 先检查列是否存在
sharpe_value = row["sharpe_ratio"] if "sharpe_ratio" in row.keys() else None
"sharpe_ratio": str(self._str_to_decimal(sharpe_value)) if sharpe_value else None,

# 方案 3: 使用三元表达式
"sharpe_ratio": str(self._str_to_decimal(row["sharpe_ratio"])) if "sharpe_ratio" in [c[0] for c in cursor.description] and row["sharpe_ratio"] else None,
```

**最佳实践**: 由于 `row.keys()` 返回列名列表，最简洁的修复是:
```python
"sharpe_ratio": str(self._str_to_decimal(row["sharpe_ratio"])) if row["sharpe_ratio"] is not None else None,
```

但前提是 SELECT 语句必须包含 `sharpe_ratio` 列。如果数据库 schema 可能滞后，应在 SQL 层面使用 `COALESCE`:
```sql
SELECT COALESCE(sharpe_ratio, NULL) as sharpe_ratio FROM backtest_reports
```

#### 【P1-重要】问题 2: 迁移脚本未添加 sharpe_ratio 列

**位置**: `scripts/migrate_to_profiles.py`

**问题描述**: 迁移脚本创建了 `config_profiles` 表，但未添加 `backtest_reports.sharpe_ratio` 列

**影响**: 新安装的系统或回滚后的系统会缺少 `sharpe_ratio` 列

**修复建议**: 添加迁移步骤

```python
async def add_sharpe_ratio_column(conn: sqlite3.Connection) -> bool:
    """添加 sharpe_ratio 列到 backtest_reports 表"""
    cursor = conn.cursor()
    
    try:
        # 检查列是否已存在
        cursor.execute("""
            PRAGMA table_info(backtest_reports)
        """)
        columns = [row[1] for row in cursor.fetchall()]
        
        if "sharpe_ratio" in columns:
            print("ℹ️  backtest_reports.sharpe_ratio 列已存在，跳过")
            return True
        
        # 添加 sharpe_ratio 列
        cursor.execute("""
            ALTER TABLE backtest_reports
            ADD COLUMN sharpe_ratio REAL
        """)
        
        print("✅ backtest_reports 表已添加 sharpe_ratio 列")
        conn.commit()
        return True
        
    except Exception as e:
        print(f"❌ 添加 sharpe_ratio 列失败：{e}")
        conn.rollback()
        return False
```
