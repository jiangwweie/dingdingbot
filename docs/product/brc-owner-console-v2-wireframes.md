# BRC Owner Console v2 原型图 / 线框图

## 1. 总体布局

Owner Console v2 采用左侧导航 + 顶部状态条 + 主内容区域。

```text
┌──────────────────┬────────────────────────────────────────────────────┐
│ 策略控制台        │  ● 只读观察中        刷新        用户 / 退出          │
│                  ├────────────────────────────────────────────────────┤
│  首页             │                                                    │
│  策略组           │                                                    │
│  执行意图         │                    页面内容                         │
│  账户订单         │                                                    │
│  复盘分析         │                                                    │
│  链路追踪         │                                                    │
│                  │                                                    │
└──────────────────┴────────────────────────────────────────────────────┘
```

```text
┌──────────────────┐
│ 策略控制台        │
│                  │
│ ● 首页            │
│   策略组          │
│   执行意图        │

│                  │
```

### 1.2 顶部状态胶囊

```text

│  ● 只读观察中        实盘只读 · 记录意图 · 禁止下单    │
```

## 3. 新建观察向导

入口：首页右上角或“当前可做”区域。

```text
[新建观察]
```

### 步骤 1：选择市场结构

```text
┌──────────────────────────────────────────────────────┐
│ 你想观察什么市场？                                     │
├──────────────────────────────────────────────────────┤
│ [趋势延续]   [震荡回归]   [波动突破]   [事件驱动]       │
└──────────────────────────────────────────────────────┘
```

### 步骤 2：选择标的

```text
┌──────────────────────────────────────────────────────┐
│ 选择关注标的                                           │
├──────────────────────────────────────────────────────┤
│ [SOL]   [ETH]   [BNB]   [BTC]                          │
└──────────────────────────────────────────────────────┘
```

### 步骤 3：系统推荐策略组

```text
┌──────────────────────────────────────────────────────┐
│ 推荐策略组                                             │
├──────────────────────────────────────────────────────┤
│ MI-001 SOL 多头                                        │
│ 适合：动量冲击后的延续观察                              │
│ 模式：只读观察 / 记录意图                                │
│ 风险：高波动，尚未进入下单阶段                            │
│                                                        │
│ [查看详情] [进入观察准备]                                │
└──────────────────────────────────────────────────────┘
```

首版可以仅显示占位，不实现完整推荐。

## 10. 截图验收清单

必须生成截图：

```text
home.png
strategy-groups.png
intents.png
account-orders.png
analysis.png
trace.png
legacy-route-redirect.png
```

截图验收要点：

```text
1. 左侧导航存在
2. 顶部状态胶囊存在
3. 首页不是旧 Command Center 风格
4. 首页一屏可见系统、账户、策略组、阻断、意图、下一步
5. 策略组页能看到策略组和具体策略
6. 执行意图页是一级页面
7. 账户订单页像交易所简化版
8. 复盘分析页不是 raw evidence dump
9. 链路追踪页是流程图样式
10. 旧路由不能打开旧页面
11. 全站无危险交易按钮
```
