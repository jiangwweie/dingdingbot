# Audit Evidence Pack

This directory contains module-level audit reports used as supporting evidence for the BRC technical debt governance program.

These reports are not current project authority and do not override:

1. Owner explicit instruction
2. Tracked code facts
3. `docs/canon/*`
4. `docs/ops/td-governance-and-refactor-plan-2026-06.md`
5. Current Sprint task card

Use these reports only as supporting context for the relevant sprint.

If an audit report conflicts with tracked code or `docs/canon/*`, tracked code and canon win.

Do not use audit reports to expand sprint scope.
Do not use audit reports to authorize execution, order placement, FinalGate changes, runtime execution, or API contract changes outside the approved sprint.
