# H3 假设分析：动态风险几何 / 退出结构

> **日期**: 2026-04-28
> **性质**: 只读分析，不跑实验，不改代码
> **前置结论**: H0 粗 regime gate 不通过, H1 SHORT-only 不进入主线, H2 0.382 limit-entry 不通过

---

## 1. H3 要解决的问题

**核心问题**: 2023 年 LONG-only Pinbar 策略亏损 -3924 USDT（WR=16.1%, Sharpe=-0.23），但 2024/2025 年盈利 +8501/+4490。如何在不破坏 2024/2025 趋势收益的前提下，压缩 2023 的亏损幅度？

**不是要解决的问题**: 让 2023 也盈利。已有证据表明 2023 是趋势环境不适配，不是参数调优能解决的。

**H3 的目标**: 找到一种"环境自适应的退出结构"，使得：
- 好环境（趋势延续强）→ 保留当前 TP=[1.0, 3.5]，最大化趋势捕获
- 坏环境（趋势延续弱）→ 更快兑现或更短 TP2，压缩单笔亏损幅度
- 不通过固定参数改变所有年份的入场或退出

## 2. H3 和 H2 的本质区别

| 维度 | H2 (0.382 limit-entry) | H3 (动态风险几何) |
|------|----------------------|------------------|
| **改变什么** | 入场方式（所有年份统一改） | 退出结构（按环境动态调整） |
| **作用对象** | 入场价格 + 止损距离 | TP 目标 + 兑现节奏 |
| **作用范围** | 固定应用于所有信号 | 仅在识别为"坏环境"时调整 |
| **对好年份的影响** | 严重削弱（missed 9 winners in 2024） | 应该无影响（好环境保持原结构） |
| **对坏年份的改善机制** | 止损距离缩小 55% → 亏损更小 | 更快兑现 → 亏损更小或盈利更早锁定 |
| **逻辑矛盾** | Fibonacci 回撤与趋势跟踪矛盾 | 无矛盾：趋势弱时更保守，趋势强时更激进 |

**本质区别**: H2 是"统一改入场"，H3 是"按环境改退出"。H2 的失败不是因为"改参数"本身不行，而是因为改的方式（固定、统一、作用于入场）与策略逻辑矛盾。H3 试图在正确的维度（退出）和正确的时机（环境识别后）做调整。

## 3. 已有文档是否支持"2023 缺 follow-through，而非入场价太差"？

**强支持**。三份独立文档一致指向同一结论：

### 3a. 2023 失效归因报告

- 入场质量: 2023 = -2.30%, 2024 = +4.24%（差异 6.54%）
- 但入场质量是 realized_pnl / entry_price 的均值，反映的是**出场结果**而非入场价格本身
- 平均持仓时长: 2023 = 17.5h, 2024 = 335.26h → 仓位更快结束 = 跟随趋势延续不足
- 结论: "趋势延续不足，仓位更容易以 SL 结束，而不是单纯的 exit 机制失效"

### 3b. 延续能力诊断报告

- MFE 均值: 2023 = 1.65%, 2024 = 3.10%, 2025 = 3.22%
- MAE 均值: 2023 = 2.41%, 2024 = 2.27%, 2025 = 2.83%（相近）
- **MFE 低 = 顺行幅度不足 = 趋势不延续**
- **MAE 正常 = 逆行幅度正常 = 入场后价格回撤幅度与好年份一致**
- 结论: "根本原因: 趋势延续性不足，而非入场质量或 exit 机制失效"

### 3c. H2 0.382 limit-entry proxy

- 入场改善仅 0.31%（微不足道）
- 2023 改善的真实原因是止损距离缩小 55%，不是入场价更好
- 这直接证明: **入场价不是 2023 亏损的主因**

### 综合判定

**2023 亏损的主因是 follow-through 不足（MFE 低），而非入场价太差（MAE 正常、入场改善微不足道）。** 这为 H3 提供了理论基础：如果问题不在入场，那就不应该改入场；应该改的是"在 follow-through 不足时如何更快兑现"。

## 4. 可选研究方向

### 4a. 固定 TP2 下调是否已经被证伪？

**已被证伪**。backtest-parameters.md 第 539 行明确记录：

> "2023 失败非出场问题：测试 4 种出场变体全部恶化，是市场环境不匹配"

4 种出场变体（包括更防守型 TP 目标）全部恶化，说明**固定下调 TP2 对所有年份统一生效**，在好年份同样削弱趋势捕获。这与 H0 粗 regime gate 和 H2 0.382 limit-entry 的失败模式完全一致：**固定参数改变无法区分好/坏环境**。

**结论**: 固定 TP2 下调已证伪，不应再碰。

### 4b. 动态 TP2 是否仍有研究价值？

**有研究价值，但需要环境分类前提**。

动态 TP2 的核心逻辑：
- 好环境 → TP2 = 3.5R（保留趋势捕获）
- 坏环境 → TP2 = 2.0R 或 1.5R（更快兑现，减少 SL 损失）

但这个逻辑的前提是**能正确识别好/坏环境**。H0 的 EMA250/EMA200 粗 regime gate 已经证明：粗粒度环境分类不够用（bull regime 下 LONG 同样亏损）。

所以动态 TP2 的研究价值取决于：
1. 是否能找到更精细的环境分类（EMA slope + ADX + 波动率状态）
2. 分类后的环境标签是否能准确预测 follow-through 强度

**当前状态**: 动态 TP2 本身未被直接测试，但固定 TP2 下调已证伪。如果环境分类能工作，动态 TP2 是合理的下一步。

### 4c. BE / trailing 是否已经被证伪？

**BE 已证伪**。backtest-parameters.md 第 537 行：

> "breakeven_enabled=False 更优：3年总 PnL 改善 +2715 USDT（+47%）"

BE=True 在所有年份都恶化，因为 TP1 后将 SL 移至入场价会过早结束仓位，减少趋势捕获空间。

**trailing 未被充分测试**。当前系统有 `trailing_stop_enabled=False`（默认关闭），但只有 BE 的对比数据，没有 trailing 的独立测试。不过：

- trailing 的逻辑是"TP1 后用 trailing stop 保护剩余仓位"
- 在好环境（趋势延续强），trailing 可能过早结束仓位（类似 BE 的问题）
- 在坏环境（趋势延续弱），trailing 可能帮助锁定已有利润
- 这与动态 TP2 的逻辑类似：**需要环境分类才能正确使用**

**结论**: BE 已证伪，不应再碰。trailing 有条件研究价值（需要环境分类前提）。

### 4d. 是否可以用 MFE/MAE/first-touch 做环境分类后再调退出？

**这是 H3 最有前景的方向**。

逻辑链：
1. 延续能力诊断报告已证明 MFE 可以区分好/坏年份（1.65% vs 3.10%）
2. 如果能在入场时（或入场后短时间内）预测 MFE 水平，就可以动态调整退出结构
3. 预测方法：
   - **入场前预测**: 用 EMA slope + ADX + 波动率状态估计趋势延续强度
   - **入场后确认**: 用 first-touch（入场后第一根 K 的方向）或 +1R reach rate 确认延续性
4. 退出调整：
   - 高 MFE 预期 → TP=[1.0, 3.5]（当前基线）
   - 低 MFE 预期 → TP=[1.0, 2.0] 或 TP1 后立即全平

**关键挑战**:
- 入场前预测的准确率是否足够？（H0 粗 regime gate 的 fill rate 约 80%，但改善不足）
- 入场后确认是否构成前瞻？（如果用入场后数据调整退出，需要确保不引入未来函数）
- first-touch / +1R reach rate 数据目前标记为 TODO，尚未计算

**结论**: MFE/MAE 环境分类 + 动态退出是最有前景的方向，但需要先解决环境分类的准确率问题。

## 6. 如果后续要跑 H3，最小 research-only 实验应该是什么？

### 实验名称: H3a — MFE 预测 + 动态 TP2 proxy

### 前置条件

1. **先跑 MFE/first-touch/+1R reach rate 计算**: 当前这些指标标记为 TODO。需要先计算每笔交易的 MFE、MAE、first-touch 方向、+1R/+2R/+3.5R reach rate，建立环境分类的数据基础。

2. **先跑环境分类准确率测试**: 用 EMA slope + ADX 组合预测 follow-through 强度，验证分类准确率是否 > 60%。

### 最小实验设计

**对照组 E0**: 当前基线（market entry, TP=[1.0, 3.5], BE=False）

**实验组 E1**: 动态 TP2 proxy
- 规则: 如果入场前 EMA slope <= 0 或 ADX < 20，则 TP2 从 3.5R 下调至 2.0R
- 否则保持 TP2 = 3.5R
- 其他参数不变

**实验组 E2**: first-touch 确认 + 动态 TP2 proxy
- 规则: 入场后看第一根 K 线方向
- 如果 first-touch 为逆行（入场后第一根 K close < entry），则 TP2 下调至 2.0R
- 否则保持 TP2 = 3.5R
- 注意: first-touch 使用入场后数据，但只看第一根 K（不构成前瞻，因为这是实时可获取的信息）

### 反前瞻保证

- EMA slope / ADX 使用入场前已收盘的 K 线数据（与 H0 v2 修正一致）
- first-touch 使用入场后第一根 K 线（实时信息，不构成前瞻）
- TP2 下调决策在入场时或入场后第一根 K 时做出（不使用未来数据）

### 输出指标

- 分年 PnL, Trades, Win Rate, Sharpe, MaxDD
- TP2 下调触发次数和占比
- TP2 下调后的 TP2 达成率 vs 基线 TP2 达成率
- 2023 改善幅度 vs 2024/2025 代价
- 环境分类准确率（预测为"弱延续"的交易中，实际 SL 占比 vs 预测为"强延续"的交易中 SL 占比）

### 预期结果

- 如果环境分类准确率 > 60%: E1/E2 应在 2023 有改善，2024/2025 代价 < 20%
- 如果环境分类准确率 < 50%: E1/E2 可能与固定 TP2 下调一样恶化好年份
- 如果 first-touch 确认有效: E2 应优于 E1（入场后确认比入场前预测更准确）

## 7. H3 假设总结

**H3 假设**: 2023 的亏损可以通过"环境自适应的退出结构"压缩，而不需要改变入场方式或固定下调所有年份的 TP 目标。

**支持证据**:
1. MFE 低（1.65% vs 3.10%）证明 2023 的问题是 follow-through 不足，不是入场价太差
2. MAE 正常（2.41% vs 2.27%）证明止损机制正常工作
3. H2 入场改善仅 0.31% 直接证明改入场不是解决方向
4. H2 止损距离缩小 55% 压缩了 2023 亏损，说明更紧风险几何有效——但不应固定应用于所有年份

**风险**:
1. 环境分类准确率可能不够（H0 粗 regime gate 已证明粗分类不行）
2. 动态退出可能引入新的参数搜索维度，增加过拟合风险
3. first-touch 使用入场后数据，需要确保不引入前瞻

**判定**: **H3 方向已关闭。** H3a 实验证明入场前特征虽有统计区分度，但绝对水平重叠导致无法设定有效阈值，与 H0/H2 失败模式一致。2023 亏损是市场环境不匹配，不是参数可调优的。建议接受 -3924 为 2024/2025 alpha 的固有成本。

**H3a 实验结果**（2026-04-28 完成）:
- 5 个特征有 >=20pp high_ft_pct spread（price_dist_ema_1h 27.9pp 最佳）
- 但 2023 B3 HFT=45% 与 2024 B1 HFT=50% 重叠——无法设阈值
- Skip-B1 过滤器：2023 改善 +19/+156，2024 恶化 -6539/-6395
- 2023 悖论：高 FT 桶 PnL 更差（B3=-1339 vs B1=-19），因为 2023 "高 FT"绝对水平仍不够
- **H3a 判定: 不通过**
