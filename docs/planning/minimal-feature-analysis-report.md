## 二、分析特征说明

### 1. 4h EMA 斜率

**定义**: 在交易入场时刻，取"上一根已闭合的 4h EMA 值"与"再往前一根已闭合的 4h EMA 值"，计算相对斜率

**计算公式**:
```
ema_slope_4h = (EMA4h_t - EMA4h_t-1) / EMA4h_t-1
```

**说明**:
- EMA4h_t = 入场时刻之前最近一根已闭合 4h K 线对应的 EMA
- EMA4h_t-1 = 再前一根已闭合 4h K 线对应的 EMA
- 如果分母为 0 或 EMA 未 ready，则该样本跳过或记为缺失

**意义**: 衡量 4h 级别的趋势强度

### 2. 价格相对 1h EMA 的距离

**定义**: 交易入场时刻，入场价格相对当前 1h EMA 的偏离百分比

**计算公式**:
```
ema_distance_1h = (entry_price - EMA1h_t) / EMA1h_t
```

**说明**:
- EMA1h_t = 该笔交易入场对应的 1h K 线时刻所使用的 EMA 值
- LONG 样本里，这个值通常为正；如果为负，也保留真实值，不要取绝对值

**意义**: 衡量入场价格相对 EMA 的偏离程度

## 七、最终结论

### ⚠️ 这两个特征里，有明显迹象可作为下一步"不开仓条件"候选

**候选特征**:
- **价格相对 1h EMA 距离**：2023 亏损交易均值为 1.7611%，与盈利交易差异明显（0.1468%）

**证据**:
1. 2023 年亏损交易的价格相对 EMA 距离（1.7611%）**高于**盈利交易（1.6143%）
2. 差异为 0.1468%，超过阈值（0.1%）
3. 说明在 2023 年，价格偏离 EMA 过远时开仓，更容易亏损

**建议**:
- 可考虑添加"价格相对 EMA 距离上限"作为不开仓条件
- 例如：当 `ema_distance_1h > 1.8%` 时，不开仓
- 需要进一步验证该阈值在 2024/2025 年的表现

**非候选特征**:
- **4h EMA 斜率**：2023 亏损交易与盈利交易差异不明显（-0.0458% < 0.1%）
- 说明 4h EMA 斜率对 2023 年的亏损交易区分度不足
