## 二、诊断指标说明

### 1. MFE（Maximum Favorable Excursion）

**定义**: 入场后 24h 内的最大顺行幅度

**计算公式**（对于 LONG）:
```
MFE = max(high - entry_price) / entry_price
```

**意义**: 衡量信号入场后的最大盈利潜力

### 2. MAE（Maximum Adverse Excursion）

**定义**: 入场后 24h 内的最大逆行幅度

**计算公式**（对于 LONG）:
```
MAE = max(entry_price - low) / entry_price
```

**意义**: 衡量信号入场后的最大风险敞口

## 五、最终结论

### ❌ 2023 年失效的主要原因：LONG 信号入场后延续性明显弱于 2024/2025 年

**核心证据**:

1. **MFE 更低（顺行幅度不足）**
   - 2023 年 MFE 均值: 1.65%
   - 2024/2025 年 MFE 均值: 3.10% / 3.22%
   - 差异: **+1.45% / +1.56%**

2. **MAE 更高（逆行幅度更大）**
   - 2023 年 MAE 均值: 2.41%
   - 2024/2025 年 MAE 均值: 2.27% / 2.83%
   - 差异: **-0.14% / +0.42%**（相近）

**结论**:
- 2023 年 LONG 信号入场后，顺行幅度明显不足（MFE 低），导致无法达到 TP 目标
- 2023 年 LONG 信号入场后，逆行幅度正常（MAE 相近），说明止损机制正常工作
- **根本原因**: 趋势延续性不足，而非入场质量或 exit 机制失效

## 七、建议

### 短期建议

1. **添加趋势环境识别**
   - 实现：使用 EMA 斜率或价格与 EMA 关系判断趋势
   - 规则：在下跌趋势中关闭 LONG-only 策略
   - 效果：避免在不适配的市场状态下开仓

2. **添加市场状态过滤**
   - 实现：使用 ATR 或波动率判断市场状态
   - 规则：在极端波动或低波动期关闭策略
   - 效果：减少在不利环境下的交易

### 中期建议

1. **多空自适应**
   - 实现：根据趋势环境自动切换 LONG/SHORT
   - 规则：上升趋势 → LONG-only，下降趋势 → SHORT-only
   - 效果：适应不同市场环境

2. **动态止损**
   - 实现：根据市场波动率动态调整止损距离
   - 规则：高波动 → 宽止损，低波动 → 窄止损
   - 效果：减少在震荡期被频繁止损
