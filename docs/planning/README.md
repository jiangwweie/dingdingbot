## 📝 使用规范

根据 `CLAUDE.md` 中的 `planning-with-files` 规范：

1. **任务启动时**: 在 `task_plan.md` 中添加任务计划
2. **发现技术洞见时**: 更新 `findings.md`
3. **每个会话结束时**: 更新 `progress.md`
4. **任务完成后**: 将详细文档移至 `archive/`
