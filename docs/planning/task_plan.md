### 数据划分

| 数据集 | 时间范围 | 用途 |
|--------|---------|------|
| 训练集 | 2022-2024（前 2 年）| Optuna 优化 |
| 测试集（OOS）| 2024-2025（最近 1 年）| 最终验证，优化期间不碰 |

### 搜索空间

| 参数 | 范围 | 当前值 | 说明 |
|------|------|--------|------|
| min_distance_pct | 0.002 ~ 0.015 | 0.005 | EMA 距离阈值 |
| tp_targets[0] | 0.8 ~ 1.5 | 1.0 | TP1 目标 |
| tp_targets[1] | 1.5 ~ 4.0 | 2.5 | TP2 目标 |
| tp_ratios[0] | 0.4 ~ 0.8 | 0.6 | TP1 仓位比例 |
| ema_period | 20 ~ 100 | 60 | EMA 周期 |

### 目标函数

```python
def objective(trial):
    """Optuna 目标函数：最大化训练集 Sharpe Ratio"""
    params = {
        "min_distance_pct": trial.suggest_float("min_distance_pct", 0.002, 0.015),
        "tp_target_0": trial.suggest_float("tp_target_0", 0.8, 1.5),
        "tp_target_1": trial.suggest_float("tp_target_1", 1.5, 4.0),
        "tp_ratio_0": trial.suggest_float("tp_ratio_0", 0.4, 0.8),
        "ema_period": trial.suggest_int("ema_period", 20, 100),
    }

    # 跑训练集回测
    report = run_backtest(train_start, train_end, params)

    # 返回 Sharpe Ratio（不是纯 PnL，防止单笔大赚掩盖风险）
    return report.sharpe_ratio
```

### 验收标准
