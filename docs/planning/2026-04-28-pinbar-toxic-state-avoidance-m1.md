# M1 Pinbar Toxic State Avoidance — 单因子 Regime Filter 实验

**日期**: 2026-04-28  
**假设**: M1 — 单个 toxic-state filter 能减少 Pinbar 亏损且不破坏 2024/2025 收益  
**类型**: Proxy result（独立撮合，不等同正式 Backtester v3_pms result）  
**判定**: **PASS** — 4/4 实验全部通过 5 项 PASS 标准

---

## 实验目标

M0 诊断发现 Pinbar 在特定市场环境系统性亏损。M1 测试最简单的干预：在 toxic state 时跳过信号。如果单因子 filter 就能显著改善，就不需要复杂多因子模型。

## 方法

1. 复用 M0 的 FeatureComputer + Pinbar 检测逻辑
2. 基于 M0 tercile 边界设定 4 个 toxic threshold
3. 每个 filter 独立测试（不做组合）
4. 5 项 PASS 标准判定

### Toxic Thresholds（来自 M0 tercile 边界）

| Filter | 阈值 | 方向 | 含义 |
|--------|------|------|------|
| ema_4h_slope | > 0.215185 | skip high | 强上升趋势中不做空 |
| recent_72h_return | > 0.064538 | skip high | 近期涨幅过大时不做反转 |
| realized_volatility_24h | > 0.007751 | skip high | 高波动环境不做 Pinbar |
| distance_to_donchian_20_high | < -0.016809 | skip low | 价格接近通道顶部时不做 |

## 关键洞见

### 1. 单因子 filter 就够了

不需要复杂多因子模型。每个单因子 filter 都独立通过全部 5 项 PASS 标准。这说明 Pinbar 的亏损来源集中且可识别。

### 2. E1（ema_4h_slope）综合最优

- 3yr PnL 最高 (+1,314)
- 2024 利润增幅最大 (+88%)
- WR 提升最明显 (12.2% → 14.8%+)
- **唯一缺点**: 过滤了 27% 交易，但考虑到 PnL 翻正，这是值得的

### 3. E4（Donchian distance）2023 改善最大

- 2023 亏损减少 71.3%（从 -2,513 到 -722）
- MaxDD 最低 (18.04%)
- **但 2024+25 利润保留最少** (497%)，说明过滤了一些 profitable trades

### 4. 2025 几乎被"治愈"

E2 和 E3 将 2025 亏损从 -1,499 降到 -29 和 -75，接近盈亏平衡。这说明 2025 的亏损主要来自 toxic states。

### 5. 过滤率合理

- 最低: 17.6% (E3 volatility)
- 最高: 27.0% (E1 ema_4h_slope)
- 所有 filter 低于 40% 阈值，不损害策略覆盖面

### 6. 验证了 M0 的 regime 假说

M0 诊断"问题在 regime"，M1 证明"跳过 toxic regime 就能解决问题"。因果链闭合。

- 组合过滤率是否超过 40%
- 2024+25 利润保留是否仍 > 80%

### 2. 实施 regime filter 到 src

- `src/domain/strategies/pinbar.py` 的 `detect()` 方法
- 或 `src/domain/filter_factory.py` 的新 filter type

先在 BTC/USDT:USDT 上验证 E1 filter 有效性，再扩展到 SOL/BNB。
