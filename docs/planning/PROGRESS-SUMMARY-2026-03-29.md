## 2026-03-29 今日完成工作汇总

### S6-2 Pinbar 评分优化与信号覆盖 ✅

**核心功能**:
1. **ATR 过滤器**: 使用 Wilder 平滑法计算 ATR，过滤低波幅 K 线
2. **评分公式**: `score = pattern_ratio × 0.7 + min(atr_ratio, 2.0) × 0.3`
3. **信号覆盖**: 时间窗口内，更优信号自动覆盖旧信号
4. **通知增强**: 覆盖通知含评分对比，反向信号含市场分歧提示
5. **前端展示**: 被覆盖信号视觉降级（灰色半透明）

**修改文件**:
- `src/application/signal_pipeline.py` - 覆盖逻辑 + 冷却缓存重建
- `src/infrastructure/notifier.py` - 通知模板增强
- `src/domain/strategy_engine.py` - ATR 值获取 + 评分公式
