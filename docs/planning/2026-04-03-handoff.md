## 遗留问题详解

### DEBT-4: asyncio.Lock 事件循环冲突 ⚠️

**问题描述**:
- `OrderRepository` 在 `__init__` 中创建 `asyncio.Lock()`
- `TestClient` 使用不同事件循环时，lock 绑定到错误事件循环
- 导致死锁：测试无法正常执行订单链完整测试

**影响范围**:
- `src/infrastructure/order_repository.py:__init__`
- 所有使用 `OrderRepository` 的测试

**建议修复方案**:
```python
class OrderRepository:
    def __init__(self, db_path: str = "data/v3_dev.db"):
        self.db_path = db_path
        self._lock = None  # ❌ 不在 __init__ 创建

    async def initialize(self):
        if self._lock is None:
            self._lock = asyncio.Lock()  # ✅ 延迟创建
```

**预计工时**: 1h
