## 🤔 常见问题

### Q1: 为什么团队角色技能不自动触发？

**原因**:
- 需要明确的**角色切换**，避免职责混乱
- 确保用户**意图明确**，知道正在使用哪个专业角色
- 不同角色有不同的**权限边界**，需要明确授权

**建议**: 复杂任务明确使用 `/coordinator` 调度，或直接指定角色（`/backend` `/frontend`）

### Q2: Memory MCP 和 PU MCP 为什么现在不能用？

**原因**: 配置修改后未重启，当前会话仍使用旧配置

**解决**: 完全退出并重启 Claude Code（`/exit` 然后 `claude`）

### Q3: 简短输入会自动触发 PUA-skill，但详细输入不触发？

**设计逻辑**:
- 简短输入（< 20 字）需要补充约束和细节 → 自动触发
- 详细输入已经足够清晰 → 不需要升级

**建议**: 需要详细提示词时主动调用 `/pua-skill`
