## 四、快速切换指南

### 方式 1：复制配置文件

```bash
# 切换到 Bybit
cp config/user.bybit.yaml.example config/user.yaml
# 编辑 API 密钥后重启

# 切换到 OKX
cp config/user.okx.yaml.example config/user.yaml
# 编辑 API 密钥后重启

# 切换回 Binance
cp config/user.yaml.example config/user.yaml
# 编辑 API 密钥后重启
```

### 方式 2：直接编辑 user.yaml

```yaml
exchange:
  name: bybit  # 修改为 bybit 或 okx
  api_key: YOUR_API_KEY
  api_secret: YOUR_API_SECRET
  testnet: false
```

## 六、注意事项

### 6.1 符号格式
系统使用 CCXT 统一格式 `BTC/USDT:USDT`（USDT 永续合约），所有交易所均采用此格式。

### 6.2 时间周期映射
| 系统格式 | Binance | Bybit | OKX |
|----------|---------|-------|-----|
| 1m | 1m | 1m | 1m |
| 15m | 15m | 15m | 15m |
| 1h | 1h | 1h | 1H |
| 4h | 4h | 4h | 4H |
| 1d | 1d | 1d | 1D |
| 1w | 1w | 1w | 1W |

CCXT 内部会自动处理时间周期格式差异，用户无需关心。

### 6.3 杠杆限制
| 交易所 | 最大杠杆 | 说明 |
|--------|----------|------|
| Binance | 125x | 主流币种 |
| Bybit | 125x | 主流币种 |
| OKX | 125x | 主流币种 |

实际可用杠杆取决于币种和账户等级。
