## 根因详细分析

### A 类问题：Repository 未初始化

**代码位置**: `src/interfaces/api.py` 第 265-271 行

```python
def _get_config_entry_repo() -> Any:
    """Get config entry repository or create a new instance if not initialized."""
    if _config_entry_repo is None:
        # Fallback: create a new instance
        from src.infrastructure.config_entry_repository import ConfigEntryRepository
        return ConfigEntryRepository()  # ❌ 未调用 initialize()
    return _config_entry_repo
```

**问题**: fallback 逻辑创建的 `ConfigEntryRepository` 实例没有调用 `await repo.initialize()`，导致 `_db` 连接为 `None`。

**影响范围**:
- `/api/strategy/params` (GET)
- `/api/strategy/params` (PUT)
- `/api/strategy/params/preview` (POST)
- `/api/strategy/params/export` (GET/POST)
- `/api/strategy/params/import` (POST)

**5 Why 分析**:
1. Why? - `_db.execute()` 被调用但 `_db=None`
2. Why? - `ConfigEntryRepository.initialize()` 未被调用
3. Why? - `_get_config_entry_repo()` fallback 返回新实例但没有初始化
4. Why? - 代码设计时假设 repository 已在全局初始化
5. Why? - 全局初始化在某些场景下失败或未执行

**修复方案**:
```python
async def _get_config_entry_repo() -> Any:
    global _config_entry_repo
    if _config_entry_repo is None:
        from src.infrastructure.config_entry_repository import ConfigEntryRepository
        _config_entry_repo = ConfigEntryRepository()
        await _config_entry_repo.initialize()  # ✅ 添加初始化
    return _config_entry_repo
```

### B 类问题：Pydantic 响应模型引用

**代码位置**: `src/interfaces/api.py` 第 2193 行、2232 行

```python
@app.get("/api/strategies/templates", response_model="StrategyTemplateListResponse")  # ❌ 字符串引用
@app.get("/api/strategies", response_model="StrategyListResponse")  # ❌ 字符串引用
```

**问题**: 使用字符串前向引用但 Pydantic 无法在运行时解析。

**错误日志**:
```
pydantic.errors.PydanticUserError: `TypeAdapter[typing.Annotated[ForwardRef('StrategyListResponse')...]]` is not fully defined
```

**修复方案**: 在文件顶部添加 `from __future__ import annotations` 并移除字符串引号：
```python
from __future__ import annotations

@app.get("/api/strategies/templates", response_model=StrategyTemplateListResponse)  # ✅ 直接引用
```

### D 类问题：错误响应格式

**代码模式**:
```python

    return {"error": str(e)}  # ❌ 缺少 error_code
```

**应为**:

```python
    return ErrorResponse(

        message=f"服务器内部错误：{str(e)}"
    )
```
