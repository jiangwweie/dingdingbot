# Project Skills

This directory contains project-specific skills for the dingdingbot project.

## Available Skills

### Architecture Roles
- `/architect` - 架构师角色，负责方案设计和代码审查
- `/reviewer` - 审查员角色，负责独立代码审查

### Agent Team (开发组)
- `/coordinator` - 团队协调器，负责任务分解和结果整合
- `/frontend` - 前端开发专家 (React + TypeScript + TailwindCSS)
- `/backend` - 后端开发专家 (Python + FastAPI + asyncio)
- `/qa` - 质量保障专家 (pytest + vitest)

## Usage

Use these skills to switch roles and reduce communication overhead when working on tasks from `docs/tasks/`.

**For full-stack feature development:** Use `/coordinator` to automatically decompose and parallelize tasks.

**For specific tasks:** Use `/frontend`, `/backend`, or `/qa` directly.
